# Metis: 학습 의도 분석 보고서

## 📋 프로젝트 요청 정보
- **프론트엔드**: Vue.js 3
- **백엔드**: Spring Boot
- **ORM**: MyBatis
- **데이터베이스**: Oracle

## 🔍 환경 스캔 결과
1. **Node.js**: ❌ 설치되어 있지 않음
2. **Java**: ❌ 확인되지 않음 (실행 결과 없음)
3. **Oracle**: ❌ 확인되지 않음

## 🎯 의도 분류
- **목적**: 실제 서비스 구축 (Production-ready)
- **복잡도**: 중급 (4개의 기술 스택 통합)
- **학습 제약 사항**: 
  - 필요한 런타임(Java, Node.js, Oracle)이 현재 시스템에 설치되지 않음
  - Oracle DB를 별도로 설치해야 함 (가능하면 Docker를 사용하거나 로컬 설치 필요)
  - Spring Boot 프로젝트 초기화를 위한 도구가 필요함

## ⚠️ 학습 제약 사항
1. **환경 설정**: Node.js, Java, Oracle이 모두 필요
2. **Oracle DB**: 로컬에 설치 필요 (Docker 가능)
3. **개발 도구**: Node.js 설치가 필요

## 📝 제안 사항
1. Node.js 설치 확인 및 설치 필요 여부 질문
2. Oracle DB 설치 방법 (Docker vs 로컬 설치)
3. Spring Boot 프로젝트 생성 방법 (Spring Initializr 사용)
